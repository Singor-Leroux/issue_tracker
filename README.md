# [Issue Tracker](https://www.freecodecamp.org/learn/quality-assurance/quality-assurance-projects/issue-tracker)

## Tech Stack

**Client:** HTML, CSS

**Server:** Node, Express

**Database:** MongoDB, Mongoose

**Testing:** Mocha, Chai

## Run Locally

```bash
git clone https://github.com/GoogolDKhan/Issue-Tracker.git

cd Issue-Tracker

npm install

npm run start
```
